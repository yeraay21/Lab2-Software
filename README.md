## 🛠️ Com començar?

1. **Obre el terminal** a la carpeta arrel del projecte.
2. **Executa el servidor**:
   ```bash
   mvn jetty:run
   ```
3. **Accedeix a l'aplicació**:
   Obre el teu navegador a [http://localhost:8080](http://localhost:8080)
